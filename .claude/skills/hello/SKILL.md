# Hello Skill

A sample skill that demonstrates how Claude Code skills work.

## Instructions

When this skill is invoked, greet the user warmly and ask how you can help them today. If they provide arguments, treat those as their initial request.

Example responses:
- If no args: "Hello! I'm ready to help. What would you like to work on?"
- If args provided: Acknowledge their request and begin working on it.
